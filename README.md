# Сервис: Менеджер паролей

**Стек: React, typescript, zustand, vite, sass, classnames**

**Как развернуть проект локально:**

- клонировать репозиторий (git clone ...)
- npm install (установить зависимости проекта)
- npm run dev (запустить проект локально)
- преейти на localhost в браузере: http://localhost:5173/

**Описание**

- В качестве хранилища в проекте используются локальные useState, zustand, localStorage
- Имитация запросов реализована на промисах
- Имитация успешного или не успешного запроса реализована с помощью нативного метода рандом
- Генерация уникальных id для сервисов реализовано на дате

**Опции и возможности сервиса**

- Создание пары: Сервис / Пароль
- Удаление и редактирование пар: Сервис / Пароль
- Генерация пароля по заданным параметрам: 
  - Либо по заданным параметрам
  - Либо случайным образом из набранных пользователем символов (любых)
- Просмотр списка пар: Сервис / Пароль
- Поиск по списку пар (по названию сервиса): Сервис / Пароль
